shell基本运算符

类别介绍

  • 算术运算符
  • 关系运算符
  • 布尔运算符
  • 字符串运算符
  • 文件测试运算符

几点注意

  • 在进行求值操作时要使用expr
  • 表达式和运算符之间必须要有空格
  • 算数表达式要完全被``包含
    a=`expr 2 + 2`
echo "两数之和为 : $a"

    算数运算符

    运算符 说明 使用举例
    + 加法 expr $a + $b
    - 减法 expr $a - $b
    * 乘法 expr $a \* $b
    / 除法 expr $a / $b
    % 取余 expr $a % $b
    = 赋值 a = $b 将变量b的值赋给a
    == 判相等，相同为true [ $a == $b ]
    != 判不等，不等为true [ $a != $b ]

注意：

  1. 乘号(*)前边必须加反斜杠(\)转义才能实现乘法运算；
  2. 条件表达式要放在方括号之间，并且要有空格，例如: [$a==$b] 是错误的，必须写成 [ $a == $b ]。

实例：

#!/bin/bash

a=10
b=20

val=`expr $a + $b`
echo "a + b : $val"

val=`expr $a - $b`
echo "a - b : $val"

val=`expr $a \* $b`
echo "a * b : $val"

val=`expr $b / $a`
echo "b / a : $val"

val=`expr $b % $a`
echo "b % a : $val"

if [ $a == $b ]
then
   echo "a 等于 b"
else
   echo "a 不等于 b"
fi

输出结果

a + b : 30
a - b : -10
a * b : 200
b / a : 2
b % a : 0
a 不等于 b

关系运算符

关系运算符只支持数字，不支持字符串，除非字符串的值是数字
假设a=20; b=30.

运算符 说明 使用举例
-eq 两个变量相等 返回 true [ $a -eq $b ] 返回false
-ne 两个变量不相等 返回 true [ $a -ne $b ] 返回true
-gt 左变量 > 右变量 返回true [ $a -gt $b ] 返回false
-lt 左变量 < 右变量 返回true [ $a -lt $b ] 返回true
-ge 左变量 >= 右变量 返回true [ $a -ge $b ] 返回false
-le 左变量 <= 右变量 返回true [ $a -le $b ] 返回true
注意
表达式要放在方括号之间，并且要有空格，[ $a -gt $b ]。

字符串运算符

假设a=“zyk”;b=“xy”.

运算符 说明 使用举例
= 两个字符串相等 返回true [ $a = $b ] 返回false
!= 两个变量不相等 返回 true [ $a ！= $b ] 返回true
z 字符串长度为0 返回true [ -z $a ] 返回false
n 字符串长度不为0 返回true [ -n $a ] 返回true
str 字符串不为空 返回true [ $a ] 返回true

文件测试运算符

file=”/shell-script/sh02.sh”

运算符 说明 使用举例
-b 文件是块设备文件 返回true [ -b $file ]返回false
-c 文件是字符设备文件 返回 true [ -c $file ]返回false
-d 文件是目录文件 返回true [ -d $file ]返回false
-f 文件是普通文件(不是目录也不是设备文件)返回 true [ -f $file ]返回true
-g 文件设置了SGID位 返回true [ -g $file ]返回false
-k 文件设置了粘着位 返回true [ -k $file]返回false
-p 文件是有名管道 返回true [ -p $file ]返回false
-u 文件设置了SUID位 返回true [ -u $file ]返回false
-r 文件可读 返回true [ -r $file ]返回 true
-w 文件可写 返回true [ -w $file ] 返回true
-x 文件可执行 返回true [ -x $file ] 返回true
-s 文件不为空 返回true [ -s $file ] 返回true
-e 文件存在 返回true [ -e $file ] 返回true

用法实例

#!/bin/bash
file="/shell-script/sh02.sh"
if [ -r $file ]
then
   echo "文件可读"
else
   echo "文件不可读"
fi
if [ -w $file ]
then
   echo "文件可写"
else
   echo "文件不可写"
fi
if [ -x $file ]
then
   echo "文件可执行"
else
   echo "文件不可执行"
fi
if [ -f $file ]
then
   echo "文件为普通文件"
else
   echo "文件为特殊文件"
fi
if [ -d $file ]
then
   echo "文件是个目录"
else
   echo "文件不是个目录"
fi
if [ -s $file ]
then
   echo "文件不为空"
else
   echo "文件为空"
fi
if [ -e $file ]
then
   echo "文件存在"
else
   echo "文件不存在"
fi

shell注释的用法

#!bin/bash
# echo "我是单行注释，你看不到我"

:<<!
echo "我是多行注释1，你看不到我"
echo "我是多行注释2，你看不到我"
!

# 其实!可以随意命名，只要别跟中间的注释内容相同即可。当Shell脚本执行遇到:<<!，就不执行脚本了，一直到再碰!后才重新开始执行脚本。如果忘记!或者写错(由于已经不执行脚本了，所以即使写错也不会报错)，则:<<!之后的脚步将都不会执行。
